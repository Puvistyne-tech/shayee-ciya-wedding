# Wedding Invitation Assets

Place the following files in this directory:

| File | Description |
|------|-------------|
| `wax-seal.png` | Full wax seal (included) |
| `broken-seal-top.png` | Top half of broken seal (included) |
| `broken-seal-bottom.png` | Bottom half of broken seal (included) |
| `intro-video.mp4` | Portrait intro video with audio |

The intro video should be a vertical (9:16) MP4 file. It will autoplay with sound after the envelope is opened.

## Couple Photos (`photos/`)

Add your photos to `public/assets/photos/` using these exact filenames:

| File | Used in | Recommended size |
|------|---------|------------------|
| `groom.jpg` | Invitation — groom portrait | 400×400 px (square) |
| `bride.jpg` | Invitation — bride portrait | 400×400 px (square) |
| `couple-hero.jpg` | We Invite You banner | 1200×600 px (wide) |
| `story-2019.jpg` | Our Story — How We Met | 800×600 px |
| `story-2020.jpg` | Our Story — First Date | 800×600 px |
| `story-2023.jpg` | Our Story — Together | 800×600 px |
| `story-2025.jpg` | Our Story — The Proposal | 800×600 px |
| `gallery-1.jpg` … `gallery-6.jpg` | Photo gallery grid | 800×1000 px (portrait) |

**Mock placeholders are included** in `photos/` — each file is labeled with its filename. Replace them with your real photos using the same names. See `photos/README.md` for details.

If a photo is missing, the site shows a graceful text placeholder until you add it.
